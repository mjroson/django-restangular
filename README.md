## django-restangular
Test rest angular with django api rest

## Install dependencies:
  $ npm install 
  $ bower install

## Run server
  $ gulp connect
  
  
## Build & development

Run `gulp build` for building and `gulp connect` for preview.

## Testing

Running `npm test` will run the unit tests with karma.
